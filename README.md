# Shruti Birthday Website

A one-page, mobile-first birthday site built with plain HTML, CSS, and JavaScript.

## Run locally

Open `index.html` in a browser.

For best results (fetch/audio behavior), serve it through a local static server:

- `python3 -m http.server 8080`
- Open `http://localhost:8080`

## Deploy on GitHub Pages

1. Push this folder to a GitHub repository.
2. Go to repository **Settings** -> **Pages**.
3. In **Build and deployment**, select **Deploy from a branch**.
4. Choose your branch (usually `main`) and folder `/ (root)`.
5. Save and wait for Pages to publish.
